# Interactive Bird Cards

An interactive card gallery showcasing beautiful bird photographs. The project is built using HTML and CSS and features smooth hover animations with a layered card effect for an engaging user experience.

## Features
- Interactive hover effects
- Layered card animation
- Clean and responsive design
- Attractive bird photo gallery
- Simple and user-friendly interface

## Technologies Used
- HTML
- CSS

## Project Preview
Hover over the cards to reveal the layered animation and explore different bird images.

## How to Run
1. Clone this repository.
2. Open the `index.html` file in your web browser.

## Future Improvements
- Add more bird categories.
- Include search and filter options.
- Enhance animations with JavaScript.
- Make the design fully responsive for all devices.
